to run this : 

1) install python (i used python 3.8) 

2)create virtual env
python -m venv tf_env

3)activate it 
tf_env\Scripts\activate

4) install falsk and tensorflow : 
pip install flask
pip install tensorflow==2.13.0

5) install flask cors and nltk
pip install flask-cors
pip install nltk

6) run app.py :
python app.py

7) access http://127.0.0.1:5000 from a browser to see result
